Grama-Sethu
AI-Powered Rural Connectivity & Flood Safety Monitoring

A real-time Android application for bridge safety monitoring
and flood reporting in rural India.

Download App
[⬇️ Download GramaSethu APK](https://github.com/ujwal-abhi/grama-sethu/raw/main/GramaSethu.apk)

Features
- 🗺️ Live Google Maps with color-coded bridge markers
- 📋 Crowd-based bridge status reporting
- 🚨 Real-time flood & danger alerts
- 🤖 AI safety assistant powered by LLaMA 3
- 👥 Community flood & road hazard reporting
- 📍 GPS location pinning for custom reports
- 🔐 Firebase Auth — Email + Google Sign-In
- 👤 User profile with report history
- ✨ Smooth animations and shimmer loading effects

Tech Stack
| Component | Technology |
|---|---|
| Language | Kotlin |
| UI Framework | Jetpack Compose |
| Architecture | MVVM + Clean Architecture |
| Database | Firebase Firestore |
| Authentication | Firebase Auth |
| Maps | Google Maps SDK |
| AI Assistant | Groq API — LLaMA 3.3 |
| Networking | OkHttp |
| Async | Kotlin Coroutines |

App Screens
- Splash Screen with animation
- Login Screen — Email & Google Sign-In
- Bridge Map Screen — Live Google Maps
- Report Screen — Known Bridge + Custom Report
- Alerts Screen — Official + Community alerts
- AI Assistant — Flood safety chat
- Profile Screen — User activity & history
